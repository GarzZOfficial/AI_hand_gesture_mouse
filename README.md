# AI_hand_gesture_mouse
I hand gester mouse project is a AI based project in which you can assess the
mouse with gestures without physically touching the mouse. This project detects the hand and counts
the fingers with the first finger up and all fingers down you can move 
the mouse and with first finger and middle finger you can click.
